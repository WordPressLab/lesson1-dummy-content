  <h1 id="list_types">List Types</h1>
   
  <h3>Definition List</h3>
  <dl>
    <dt>Definition List Title</dt>
    <dd>This is a definition list division.</dd>
  </dl>
   
  <h3>Ordered List</h3>
  <ol>
    <li>List Item 1</li>
    <li>List Item 2</li>
      <ol>
        <li>sub-List Item 1</li>
        <li>sub-List Item 2</li>
      </ol>
    <li>List Item 3</li>
  </ol>
  <p>You can change the style of list numbers (like roman numerals, letters, etc.) using a CSS rule. Check <a target="blank" href="http://www.w3schools.com/tags/tag_ol.asp"> W3Schools</a> for a full list.</p>
  <h3>Unordered List</h3>
  <ul>
    <li>List Item 1</li>
    <li>List Item 2</li>
      <ul>
        <li>sub-List Item 1</li>
        <li>sub-List Item 2</li>
      </ul>
    <li>List Item 3</li>
  </ul>
  <p>You can change the style of list bullets using a CSS rule. Check <a target="_blank" href="http://www.w3schools.com/tags/tag_ul.asp">W3 Schools</a> for a full list. You can even use your own image for bullets if you like using the list-style-image attribute</p>

  <h1 id="misc">Misc Stuff - abbr, acronym, code, sub, sup, etc.</h1>
   
  <p>Like with exponents you have <sup>superscript</sup> or logarythms you have <sub>subscript</sub>. The <code>&lt;acronym&gt;</code> tag makes a tooltip pop up over an acronym as in <acronym title="Global Information Internship Program">GIIP</acronym>. There is also the abbreviation tag that functionally does the same thing as acronym. For example <abbr title="Avenue">AVE</abbr> is done with the <code>&lt;abbr&gt;</code> tag. It does the exact same thing as the <code>&lt;acronym&gt;</code> tag. Not sure why that is...</p> 
  
  <p><b>Fun Fact:</b>A series of words shortened to their first letters, as in <acronym titpe="University of California Santa Cruz">UCSC</acronym>, is called an <i>Initialism</i>. It is only an <i>Acronym</i> if it is pronounced as a word, as in <acronym titpe="Global Information Internship Program">GIIP</acronym>.</p>
